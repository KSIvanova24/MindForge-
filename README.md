## 📌 About

MindForge is a task management app that helps you stay organized.

You can:
- create tasks
- set priorities
- add deadlines
- split tasks into smaller steps
- track progress

---

## ❗ Problem

People often:
- forget tasks
- miss deadlines
- feel overwhelmed
- don’t know where to start

---

## 💡 Solution

MindForge helps you:
- know what to do next
- focus on important tasks
- track your progress
- stay organized

---

## 🚀 Features

- Task management (create, edit, delete)
- Priority system
- Deadlines
- Subtasks
- Progress tracking
- Clean dashboard
- Login system
